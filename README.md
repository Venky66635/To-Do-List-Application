## 📝 To-Do List Application – Python

A simple command-line based To-Do List application built using Python that allows users to manage daily tasks efficiently.

The application enables users to add, view, mark as complete, and delete tasks, with all data stored in a file to ensure persistence between sessions.

### 🚀 Features

* Add new tasks
* View all tasks
* Mark tasks as completed
* Delete tasks
* Persistent storage using file handling
* Clean and structured command-line interface

### 🛠️ Technologies Used

* Python (Core concepts only)

This project demonstrates the practical use of lists, dictionaries, control flow, and file handling while reinforcing logical thinking and modular code design.
